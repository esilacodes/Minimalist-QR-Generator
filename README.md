# 🚀 QuickQR Creator

Hey! Welcome to **QuickQR Creator**, a lightweight and minimalist desktop tool built with Python to make QR code generation as simple as it should be. No web browsers, no ads—just run it, type your link, and get your QR.

---

## ✨ Why This Project?
I built this because I wanted a clean, distraction-free way to generate QR codes right from my desktop. It’s designed with a "Black & White" aesthetic that looks great on any OS and follows **Object-Oriented Programming (OOP)** principles under the hood to keep the code as clean as the UI.

### 🌟 Features
- **Modern Minimalist UI**: A bespoke dark-mode-inspired interface.
- **Smart Organization**: Automatically saves everything into a `/qrcodes` folder.
- **Robust Logic**: Built with error handling to catch empty inputs or system issues.
- **Privacy First**: Everything happens locally on your machine.

## 🛠️ Tech Stack
- **Core:** Python 3.10+
- **Interface:** Tkinter (Standard Library)
- **Engine:** qrcode & Pillow
- **Architecture:** Class-based OOP

1. **Setup and Run:**
   ```bash

   git clone [https://github.com/esilacodes/Minimalist-QR-Generator.git]

   # Install dependencies
   pip install -r requirements.txt

   # Run the application
   python main.py
